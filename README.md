# OwnWarehouse
> OwnWarehouse aims to be a free, open-source and lightweight solution to store and organize your belonglings at home.

![Banner](images/Banner.png)

## Current Features
- Fancy GUI
- Search for items
- Filter items
- History of Items
- Automatic location-algorithm
